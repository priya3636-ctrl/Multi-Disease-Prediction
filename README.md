# Multi-Disease-Prediction
MLP is a Neural Network model that can learn complex relationships between symptoms and diseases. Since our dataset contains many symptom features, MLP was able to capture hidden patterns better than traditional machine learning algorithms, resulting in higher accuracy.
# Multi-Disease Prediction System

## Project Overview
This project predicts diseases based on patient symptoms using Machine Learning.

## Dataset
Community Healthcare MultiSymptoms Disease Diagnostic Dataset

## Models Used
- Random Forest
- Logistic Regression
- KNN
- SVM
- MLP

## Results

| Model | Accuracy |
|---------|---------|
| Random Forest | 42.84% |
| Logistic Regression | 42.60% |
| KNN | 30.04% |
| SVM | 41.89% |
| MLP | 55.03% |

## Best Model
MLP Neural Network
